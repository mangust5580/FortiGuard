# FortiGuard Portfolio project — multipage SaaS landing page for a fictional cybersecurity platform.

**Live demo:** https://mangust5580.github.io/FortiGuard/

---

## Demo note

FortiGuard is a portfolio/demo project. All company details, addresses, phone numbers, email addresses, articles, cases and legal texts are fictional and used only to demonstrate the frontend implementation.

Forms do not send real data. There is no backend, form service, client data or NDA content. Successful demo/consultation submits show an in-modal confirmation state instead of navigating to a separate thank-you page.

## Tech stack

- **HTML** — modular structure via partials/templates
- **SCSS** — BEM methodology, design tokens and CSS custom properties
- **JavaScript** — vanilla ESM modules bundled with esbuild
- **Build** — Gulp 5
- **Images** — responsive AVIF/WebP/PNG output
- **Quality** — ESLint, Stylelint, Prettier
- **Deploy** — GitHub Pages via GitHub Actions

---

## Getting started

**Requirements:** Node.js >= 20

```bash
npm ci
npm run dev
```

Open the local dev server shown in the terminal.

```bash
npm run build   # production build → public/
npm run preview # build + local preview server
npm run check   # ESLint + Stylelint + production build
```

---

## Deployment

GitHub Actions builds and deploys the site from `main`.

- Workflow: `.github/workflows/deploy.yml`
- Install: `npm ci`
- Quality gate: `npm run check`
- Output: `public/`
- Pages source: **GitHub Actions**

The `public/` directory is generated by CI and is not committed.

## Project structure

```text
src/
  pages/          # Entry HTML files
  shared/         # Head, layout, sections, components, forms, modals
  styles/         # SCSS entry, helpers, tokens, layouts, components, sections
  scripts/        # ESM entry, modules, config, utils
  assets/         # Source images, icons, fonts, favicons
  static/         # Static files copied to public/
config/           # Build configuration
gulp/             # Gulp 5 build pipeline
gulpfile.js       # Gulp public entry
```
